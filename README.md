# R-Beginners 📊

Welcome to **R-Beginners**, your complete guide to getting started with R programming and data analysis! This repository contains a carefully curated collection of tutorials, guides, and resources designed to help beginners learn R from the ground up.

## Overview

This repository is designed for beginners who want to learn R programming and data analysis. Whether you're completely new to programming or transitioning from another language, you'll find resources here to guide you through each step of your learning journey.

## Getting Started

### Quick Start
1. Install R from [r-project.org](https://www.r-project.org/)
2. Install RStudio from [rstudio.com](https://www.rstudio.com/products/rstudio/download/)
3. Review the guides in this repository to start learning

## Repository Contents

### 📖 Installation & Setup
- **R_Installation_Guide.docx** - Step-by-step instructions for installing R on Windows, macOS, and Linux
- **RStudio_Interface_Guide.pptx** - A visual guide to navigating the RStudio interface and understanding its key components

### 🎓 Core Concepts
- **R_Programming_Overview.docx** - An introduction to R programming fundamentals, syntax, and basic concepts
- **r_dataframes.pptx** - Comprehensive guide to working with dataframes, the fundamental data structure in R
- **R_Data_Import_Guide.pptx** - Learn how to import data from various file formats (CSV, Excel, etc.) into R

### 🛠️ Data Manipulation & Transformation
- **dplyr_tidyr_workshop_guide.docx** - Master data manipulation using the popular dplyr and tidyr packages:
  - Filtering and selecting data
  - Creating new variables
  - Summarizing and grouping data
  - Reshaping data

### 📊 Data Visualization
- **ggplot2_guide.docx** - Create beautiful and informative visualizations using ggplot2:
  - Scatter plots, bar charts, histograms
  - Customizing colors, themes, and facets
  - Publication-ready graphics

### 📋 Sample Data
- **sales_data.csv** - sample dataset for **Data Visualization**
- **departments.csv** - sample dataset for **data manipulation and analysis**
- **employees.csv** - sample dataset for **data manipulation and analysis**
- **quarterly_scores** - sample dataset for **data manipulation and analysis**

## Learning Path

We recommend following this sequence for optimal learning:

1. **Start Here**: R_Installation_Guide.docx & RStudio_Interface_Guide.pptx
2. **Fundamentals**: R_Programming_Overview.docx
3. **Data Structures**: r_dataframes.pptx
4. **Getting Data**: R_Data_Import_Guide.pptx
5. **Manipulating Data**: dplyr_tidyr_workshop_guide.docx
6. **Visualizing Data**: ggplot2_guide.docx

Practice with the included `sales_data.csv` file as you work through each guide!
